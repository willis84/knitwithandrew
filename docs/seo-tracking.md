# SEO Tracking

## Search Console

1. Add and verify `https://knitwithandrew.com/` in Google Search Console.
2. Submit `https://knitwithandrew.com/sitemap.xml`.
3. Check Coverage and Page indexing weekly.
4. Watch the Performance report for these queries:
   - knitting
   - knitting tutorials
   - knitting patterns
   - knitting blog
   - pattern testing

## Bing Webmaster Tools

1. Add and verify `https://knitwithandrew.com/` in Bing Webmaster Tools.
2. Submit `https://knitwithandrew.com/sitemap.xml`.
3. Compare indexed pages against the live public routes and published posts.

## What To Track

- Indexed page count for homepage, blog, guides, patterns, pattern testing, and about.
- Query impressions and clicks for the target search terms.
- Clicks on Instagram CTAs via the `data-cta="instagram-follow"` attribute.
- Clicks on newsletter placeholders via the `data-cta="newsletter-coming-soon"` attribute once analytics are enabled.

## Analytics Guidance

No invasive analytics are set up by default.

If analytics are added later, prefer one of these:

- Plausible
- Fathom
- Cloudflare Web Analytics

## Review Rhythm

- Weekly: check indexing errors and search queries.
- Monthly: review titles, descriptions, and pages with the best impressions but weak click-through rate.
- After publishing a new post: request indexing and confirm it appears in the sitemap and RSS feed.
